React 整合 - useGSAP

useGSAP 是 GSAP 官方提供的 React Hook，自動處理動畫清理和生命週期管理。

安裝方式：

npm install @gsap/react

基本用法：

import { useGSAP } from "@gsap/react"; import gsap from "gsap"; function MyComponent() { const container = useRef(); useGSAP(() => { gsap.to(".box", { rotation: 360, duration: 2, repeat: -1 }); }, { scope: container }); return <div ref={container}> <div className="box" /> </div>; }

進階特性：

  • ✓ 自動清理動畫，防止記憶體洩漏
  • ✓ 支援依賴陣列，類似 useEffect
  • ✓ 範圍限定 (scope) 功能
  • ✓ 支援 Context 安全選擇器

提示：useGSAP 會在組件卸載時自動調用 kill() 方法， 確保所有動畫都被正確清理。
